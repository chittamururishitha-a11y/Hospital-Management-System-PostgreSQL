
# 🏥 Hospital Management System (PostgreSQL)

## 📌 Project Overview

The **Hospital Management System** is a SQL-based database project developed using **PostgreSQL**. It is designed to manage hospital operations such as patient registration, doctor management, appointments, prescriptions, medicines, billing, and reports.

This project demonstrates advanced SQL concepts including joins, aggregate functions, subqueries, views, stored functions, stored procedures, triggers, and indexes.

---

## 🎯 Objectives

* Manage patient records
* Manage doctor information
* Schedule appointments
* Store prescriptions
* Maintain medicine inventory
* Generate bills
* Produce analytical reports
* Improve query performance using indexes

## ▶️ How to Run the Project

### Step 1

Create a PostgreSQL database.


Execute the files in the following order:

1. create_tables.sql
